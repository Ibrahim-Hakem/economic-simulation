
# Title : Economic Simulation
It is a simple economic simulation where individuals produce and consumes while making the most rational choice to maximise their profit. There are a set number of people, a need to eat and live on a house, while some can become entrepreneurs. There is a central bank which distribute and prints money.



## Why making this project ?
I am Ibrahim, I want to make this project to illustrate in practice my curent economic science teachings. Like the law of offer and demand regulated by the market, or the impact of inflation on individuals exchanges.

It needs to remain simple enough to show directly what's happening.


## How is it made
It's made with the Mesa python library that perform the model with its agents.
